# PopUpEmpire.wtf — Taskade Foundation Blueprint

This blueprint sets up Taskade around PopUpEmpire.wtf’s core verticals:

- **Shop_wtf**
- **Blog_wtf**
- **Hack_wtf**
- **Club_wtf**

## Mission Anchor

Build a cool digital clubhouse for misunderstood adults navigating neurotypical worlds, with community, accountability, tools, wisdom, and socially awkward navigation support.

## Primary Operator Profile

- **RC**: human female solopreneur
- AI-assisted execution model focused on practical, repeatable delivery

## AI Operating Model

- **Manus = workhorse executor**
  - Runs task execution, checklist completion, follow-through, and daily operations.
- **Claude = strategy + reflection partner**
  - Handles planning quality, narrative clarity, idea shaping, and review feedback.
- **Loop**
  1. Claude proposes/refines plan.
  2. Manus executes tasks in Taskade.
  3. Claude reviews outcomes and adjusts direction.

## Vertical-by-Vertical Foundation

### Shop_wtf
- Printable worksheets, planners, backup plans, reminders, hobby packs.
- Apparel and merch that “say what you can’t.”
- Product pipeline: idea → draft → test drop → feedback → iterate.

### Blog_wtf
- Wisdom library and story-driven posts.
- Guides for adulting systems, emotional regulation, social navigation, and recovery plans.
- Editorial flow: topic queue → outline → draft → publish → repurpose.

### Hack_wtf
- Tools, templates, and automations for ADHD/neurodivergent-friendly execution.
- Internal experiments for workflows, prompts, trackers, and lightweight systems.
- Build loop: pain point → prototype → test with community → productize.

### Club_wtf
- Community, accountability, quests, and gamified self-care.
- Nostalgic arts/crafts/fun challenge tracks.
- Member loop: join → onboarding quest → weekly check-ins → milestone rewards.

## Recommended Taskade Workspace Structure

Create a top-level workspace or folder: **PopUpEmpire.wtf HQ**

Subfolders:
1. `01-Strategy-and-Vision`
2. `02-Shop_wtf`
3. `03-Blog_wtf`
4. `04-Hack_wtf`
5. `05-Club_wtf`

In each vertical folder, start with:
- `Backlog`
- `Current Sprint`
- `Content/Product Calendar`
- `Launch Checklist`
- `Metrics and Learnings`

## Core Program Tracks

1. **Adulting Support**  
   Systems that help adults adult in neurotypical environments.
2. **Printable Utility Products**  
   Worksheets, planners, reminders, fallback plans.
3. **Expressive Apparel**  
   Wearable communication for hard-to-say feelings/thoughts.
4. **Gamified Accountability**  
   Streaks, quests, points, and milestone rituals.
5. **Creative Nostalgia**  
   Arts, crafts, playful quests, and restorative fun.

## Weekly Execution Cadence

- **Monday (Claude-led planning):** prioritize goals and define sprint outcomes.
- **Daily (Manus-led execution):** complete top 3 tasks, update blockers, ship one artifact.
- **Friday (Claude + Manus review):** summarize wins, lessons, and next experiments.

## Definition of Done (Foundation Phase)

- All four vertical folders exist with baseline projects.
- Manus task queue is active for each vertical.
- Claude review checkpoint is attached to each weekly sprint.
- First publishable asset exists in each vertical.
